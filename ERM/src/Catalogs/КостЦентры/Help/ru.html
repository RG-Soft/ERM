Справочник "Accounting units"

Справочник "Accounting units" (AUs) предназначен для хранения списка используемых кост-центров.

Реквизиты шапки

  • Code (строка) - код.

  • Name (строка) — имя.

  • For deletion (булево) - признак того, что данный кост-центр будет удален.

  • Sub sub segment (справочник Segments)

  • Tax registration (справочник Divisions) - налоговая регистрация.

  • Default activity (строка) - активити, используемое по-умолчанию для данного кост-центра.
